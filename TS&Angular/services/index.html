• Const
    • Часто используется для конфигурации по умолчанию в директивах
  • Value
    • Сходство с константой, но присутствует возможность изменения
    • Часто используется для настройки директив
  • Service
    • Работает так же как фабрика
    • Использует конструктор, поэтому, при первом выполнении, выполнит new для создания экземпляра объекта
  • Factory
    • Данный сервис может вернуть любой тип данных
    • Не содержит правил по созданию этих самых данных
    • Фабрика должна только что-то вернуть
  • Provider
    • Работает так же как фабрика
    • Можно настроить в конфигурационной функции
    • Единственная служба, которую вы можете передать в вашу .config() функцию
