Замыкание - это функция и все внешние переменные, которые ей доступны. Также это «вложенные» функции в JavaScript

  • Позволяет сохранять некие промежуточные значения
  • Инкапсуляция определённых данных
  • Часто используются для назначения обработчиков событий

this

Использование this гарантирует, что функция работает именно с тем объектом, в контексте которого вызвана.

Если одну и ту же функцию запускать в контексте разных объектов, она будет получать разный this.

Отличие function(){} от () => {}

()=>{} - это краткий вариант создания функции.

  • Нельзя использовать в качестве конструктора
  • Нет своего this
  • Используют аргументы внешней "стандартной" функции
